# Project run and installation

Just make executable

``
chmod +x run.sh
``

execute run.sh

``
./run.sh
``

and input:

1 - for project run

2 - for tests

3 - for view code coverage

then press enter

If you selected 1, you also can change path to file or keep it and press enter again

## Application settings

Please check
`src/DI/variables.php`
